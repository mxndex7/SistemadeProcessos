## SPFCAS — Sistema de Processos (Fundação CAS)

Aplicação front-end em React + Vite para gerenciamento de processos (estilo SEI).

## Executar localmente.

Prerequisitos: `Node.js` 16+ e `npm`

1. Instale dependências:
   `npm install`
2. Rode em modo de desenvolvimento:
   `npm run dev`

Observação: este repositório foi adaptado para uso local. Linhas específicas de integração com AI Studio foram removidas.
